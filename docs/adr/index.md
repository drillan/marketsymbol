# アーキテクチャ決定記録 (ADR)

このセクションには marketsymbol プロジェクトのアーキテクチャ決定記録 (ADR) が含まれています。

ADR はプロジェクト開発中に行われた重要なアーキテクチャ上の決定を文書化します。

## ADR フォーマット

各 ADR は標準フォーマットに従います：

- **Status**: Draft, Proposed, Accepted, Deprecated, Superseded
- **Date**: 決定が行われた日付
- **Context**: 背景と対処する問題
- **Decision Drivers**: 決定に影響を与える主要な要因
- **Considered Options**: 評価された選択肢
- **Decision**: 選択されたアプローチ
- **Consequences**: 良い影響、悪い影響、中立的な結果
